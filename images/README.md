## Image Specifications

**Recommended sizes:**
- Portraits: 300x300px (square)
- Team photo: 1200x800px (landscape)

**Format:** JPG or PNG
**File size:** Keep under 500KB for fast loading

## How to Add Images

2. Upload them to this folder in your GitHub repository
