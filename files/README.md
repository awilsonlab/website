# Files Folder

This folder contains downloadable files referenced in the Interactive Datasets page.

## Required Files

From your current website, you'll need to upload:

### Abo, Merritt et al. Dataset
- `Abo-Merritt-et-al-methods.docx`
- `AATD_scRNAseq_R.txt`
- `CellIdentity_Genes.xlsx`
- `FGSEA_AATD_AT2s.xlsx`
- `FGSEA_AATD_AMs.xlsx`

## How to Add Files

1. Download the files from your current BU website
2. Upload them to this folder in your GitHub repository
3. The links on the datasets page will work automatically

## Note

The external links (Shiny apps, SPRING viewers) will continue to work as they're hosted elsewhere. You only need to upload the downloadable files listed above.
